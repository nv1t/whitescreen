# Screen Light

A minimal single-page web app that turns any external monitor into a coloured light source. Pick a colour, set the brightness, go fullscreen — done.

**[Live demo →](https://nuitg.github.io/whitescreen)** <!-- update with your actual GitHub Pages URL -->

---

## Features

- **Colour presets** — white, warm white, yellow, orange, red, green, cyan, blue, purple, pink, black
- **Custom colour** — browser colour picker or type any `#rrggbb` hex value directly
- **Brightness slider** — dim the light without changing the hue
- **Fullscreen** — one click to fill the whole screen; toolbar auto-hides on entry
- **Toggle controls** — 🎨 button (bottom-left) always visible, click to show/hide the toolbar
- **Keyboard shortcut** — press `H` to toggle the toolbar
- **Double-click** the background to toggle the toolbar

## Hosting on GitHub Pages

1. Fork or push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to your `main` branch, root `/`
4. Your page will be live at `https://<username>.github.io/<repo>/`

## Usage

1. Open the page on the monitor you want to use as a light
2. Pick a colour from the presets or enter a custom hex value
3. Adjust brightness with the slider
4. Click **⛶ Fullscreen** to fill the screen
5. The toolbar hides automatically — click 🎨 to bring it back
